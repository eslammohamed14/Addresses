# Lean Node - React Native Task

## Installation Guide

Make sure that `npm` and `node` are installed on your system.

#### System Requirements

```
- node v14.19.1
- npm v6.14.16
- buildToolsVersion = "31.0.0"
- minSdkVersion = 21
- compileSdkVersion = 31
- targetSdkVersion = 31
```

#### Running the App

1. Run `npm i`
2. Run `npm start`
3. Run `react-native run-android`
